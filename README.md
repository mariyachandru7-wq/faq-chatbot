FAQ Chatbot using NLP

📌 Project Description

This project is a simple FAQ Chatbot built using Python and Natural Language Processing (NLP). It matches user questions with predefined FAQs and returns the most relevant answer using cosine similarity.

🚀 Features

- Matches user queries with stored questions
- Uses TF-IDF Vectorization
- Calculates similarity using cosine similarity
- Simple command-line chatbot
- Continuous chat until user exits

🛠️ Technologies Used

- Python
- Scikit-learn (TfidfVectorizer, cosine_similarity)

▶️ How to Run

1. Install required library:
   pip install scikit-learn

2. Run the program:
   python chatbot.py

📂 Project Files

- chatbot.py → Main chatbot program
- README.md → Project description

💡 How It Works

- The chatbot stores predefined questions and answers
- Converts questions into numerical vectors using TF-IDF
- Compares user input with stored questions
- Finds the most similar question using cosine similarity
- Returns the corresponding answer
